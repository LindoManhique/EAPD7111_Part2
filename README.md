# TECHMOVES - CONTRACTS & SERVICE MANAGEMENT SYSTEM

## PROJECT OVERVIEW

TECHMOVES IS A FULL STACK ASP.NET CORE MVC APPLICATION DESIGNED TO MANAGE CLIENTS, CONTRACTS, AND SERVICE REQUESTS. THE SYSTEM INCLUDES FILE UPLOADS, BUSINESS RULE VALIDATION, CURRENCY CONVERSION, AND AUTOMATED TESTING USING GITHUB ACTIONS.

THE APPLICATION IS BUILT USING:
- ASP.NET CORE MVC (.NET 8)
- ENTITY FRAMEWORK CORE
- SQL SERVER
- XUNIT UNIT TESTING
- GITHUB ACTIONS CI/CD PIPELINE
- BOOTSTRAP FRONT-END UI

---

## CORE FEATURES

### CLIENT MANAGEMENT
- CREATE CLIENTS
- VIEW CLIENT LIST
- UPDATE CLIENT DETAILS
- DELETE CLIENTS

---

### CONTRACT MANAGEMENT
- CREATE CONTRACTS WITH:
  - CLIENT LINKING
  - START AND END DATES
  - STATUS (DRAFT, ACTIVE, EXPIRED, ON HOLD)
  - SERVICE LEVEL AGREEMENTS
  - PDF AGREEMENT UPLOAD (VALIDATED)
- EDIT CONTRACTS
- DELETE CONTRACTS
- FILTER CONTRACTS BY:
  - STATUS
  - DATE RANGE
- DOWNLOAD CONTRACT AGREEMENT PDF

---

### SERVICE REQUEST MANAGEMENT
- CREATE SERVICE REQUESTS LINKED TO CONTRACTS
- VIEW SERVICE REQUEST DETAILS
- EDIT AND DELETE REQUESTS
- BUSINESS RULE VALIDATION:
  - SERVICE REQUESTS CANNOT BE CREATED IF CONTRACT IS EXPIRED OR ON HOLD

---

### CURRENCY CONVERSION SERVICE
- USD TO ZAR CONVERSION USING EXTERNAL API
- RESPONSE CACHING FOR PERFORMANCE OPTIMISATION
- FALLBACK RATE FOR OFFLINE RELIABILITY
- UNIT TESTED BUSINESS LOGIC

---

### FILE UPLOAD VALIDATION
- ONLY PDF FILES ARE ALLOWED FOR CONTRACT AGREEMENTS
- INVALID FILE TYPES RETURN ERROR MESSAGE:
  "ONLY PDF FILES ALLOWED"
